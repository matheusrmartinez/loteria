## Installation

Install loteria with yarn

```
    # Clone reporitory
    $ git clone https://github.com/matheusrmartinez/loteria

    # Access the repo
    $ cd loteria
    
    # Install all the dependencies
    $ yarn
    
    # Start the project
    $ yarn start
    
```
